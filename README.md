## About

A personal developer portfolio built around a simple idea: it looks and
behaves like the financial systems I actually build — a ledger/terminal
aesthetic instead of a generic template. Includes a working AI chat widget
that answers questions about my background in real time.

**Highlights:**
- 🖥️ Terminal-style animated hero intro
- 📊 Experience section styled as a running "ledger"
- 🤖 AI chat widget — ask it about my work, backed by the Anthropic API
- 📱 Fully responsive, accessible, respects reduced-motion preferences
- ⚡ Built with React + Vite, deployed on Vercel

## Tech Stack

`React` · `Vite` · `JavaScript (ES6+)` · `CSS3` · `Anthropic API` (serverless)
